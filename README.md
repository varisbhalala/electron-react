#### Socketeer: A simple sockets app made with Electron

##### Server (made using Backpack)
To run:
- `cd server`
- `npm run dev`

To build:
- `cd server`
- `npm run build`

##### Client (made with React using CRA and `customize-cra`)
To run:
- `npm run electron-dev`

To build:
- `npm run electron-pack`
